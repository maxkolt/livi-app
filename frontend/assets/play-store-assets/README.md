# Инструкция по подготовке изображений для Google Play Store

## Что нужно сделать:

### 1. Сохраните ваши скриншоты
Сохраните все скриншоты приложения в эту папку (`play-store-assets`)

### 2. Создайте баннер для описания (1024x500)

**Вариант A: Использовать HTML баннер**
1. Откройте файл `feature-graphic.html` в браузере
2. Сделайте скриншот точно 1024x500px
3. Сохраните как `feature-graphic.png`

**Вариант B: Создать в редакторе**
- Используйте Canva, Figma или другой редактор
- Размер: 1024x500 px
- Формат: PNG или JPEG
- Размер файла: до 15 МБ

### 3. Обработайте скриншоты

После того как вы сохраните скриншоты в эту папку, запустите:

```bash
cd /Users/maximkoltovich/LiVi/livi-app/frontend/assets/play-store-assets
./process_images.sh
```

Скрипт автоматически:
- Создаст баннер из логотипа (если есть)
- Обработает скриншоты для смартфона (9:16, 1080x1920)
- Создаст версии для 7-дюймовых планшетов
- Создаст версии для 10-дюймовых планшетов (1440x2560)

### 4. Ручная обработка скриншотов (если скрипт не работает)

Для каждого скриншота выполните:

**Для смартфона (9:16):**
```bash
sips -z 1920 1080 ваш-скриншот.png --out phone-1.png
```

**Для 10-дюймовых планшетов:**
```bash
sips -z 2560 1440 ваш-скриншот.png --out tablet10-1.png
```

## Требования:

### Картинка для описания:
- Размер: 1024x500 px
- Формат: PNG или JPEG
- Размер файла: до 15 МБ

### Скриншоты для смартфона:
- Минимум: 2 скриншота
- Соотношение: 16:9 или 9:16
- Размер: от 320 до 3840 пкс
- Формат: PNG или JPEG
- Размер файла: до 8 МБ каждый

### Скриншоты для планшетов:
- Те же требования, что и для смартфона
- Для 10-дюймовых: от 1080 до 7680 пкс
